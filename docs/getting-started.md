# Getting Started

Overview of ACS Network Monitoring Platform

ACS Monitor is a comprehensive network and infrastructure monitoring platform that provides real-time visibility into your SNMP devices, service endpoints, and network health. It combines SNMP polling, multi-protocol service monitoring, webhook-based alerting, and rich dashboard visualization into a single unified interface.

#### SNMP Monitoring

Poll SNMP v1/v2c/v3 devices, track OIDs, upload MIBs, receive traps, and set metric-based alert rules with thresholds.

#### Service Monitors

Monitor HTTP/S, TCP, SSH, SMTP, FTP, POP3, IMAP, DNS, MySQL, and Ping endpoints with configurable intervals and alerting.

#### Alerting & Notifications

Multi-channel alerts via Email, SMS (Twilio), Slack webhooks, and custom webhooks. Escalation rules, cooldowns, and recovery notifications.

### Key Capabilities

- **Unified Device Hub** — Single view combining SNMP data, service monitors, webhooks, and alerts per device

- **25 Dashboard Widget Types** — Line charts, gauges, status grids, device health, heatmaps, alert feeds, and more

- **Network Diagnostic Tools** — Ping, Traceroute, Port Scanner, SSL Check, DNS Lookup, WHOIS, HTTP Headers

- **Auto-Discovery** — Scan subnets via SNMP to find and import devices automatically

- **MIB Management** — Upload, parse, and auto-resolve OID names from MIB files

- **SNMP Trap Receiver** — Receive and log SNMP traps with configurable filtering

- **Webhook Ingest** — Receive data from external systems via unique webhook URLs per device

- **Role-Based Access Control** — Four roles (Super Admin, Admin, Operator, Viewer) with granular permissions

- **Two-Factor Authentication** — TOTP-based 2FA with recovery codes

- **REST API** — Full CRUD API with bearer token authentication for all resources

- **Real-Time Updates** — WebSocket-powered live data push
