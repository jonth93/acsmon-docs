# Keyboard Shortcuts

## Keyboard Shortcuts & Tips

Productivity shortcuts for power users

Global Search
Ctrl + K

Toggle Auto-Refresh
Click refresh interval selector in top bar

Switch Dashboard
Dashboard selector dropdown in top bar

Acknowledge Alert
Click acknowledge button on alert event

### Tips

- Set a refresh interval (top bar) to get real-time updates on any page

- Use the global search to quickly find any device, monitor, or alert by name or IP

- Bulk operations: select items with checkboxes, then use bulk action buttons

- Dashboard widgets auto-refresh based on the global refresh interval

- The notification bell updates in real-time — new alerts appear without refreshing
